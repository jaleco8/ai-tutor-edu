# Mobile — React Native (Expo)

App para tablets Android económicas.

## Setup

```bash
npx expo install
npx expo start
```

## Variables de entorno

```
EXPO_PUBLIC_API_URL=
```

## Pantallas planeadas

- Selección de materia (Math / Code / English)
- Pantalla del tutor (ejercicio + chat con IA)
- Progreso del alumno
